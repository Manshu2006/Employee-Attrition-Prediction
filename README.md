# Employee Attrition Prediction 

Employee Attrition Prediction is a Machine Learning project that predicts the likelihood of an employee leaving an organization based on various workplace, demographic, and performance-related factors. The project helps HR teams identify employees at risk of attrition and take proactive measures to improve retention.

## Features
Employee attrition prediction using Machine Learning

Interactive Streamlit web application

Real-time prediction with user-friendly input forms

Attrition probability score visualization using a gauge chart

Employee profile analysis based on HR metrics

Clean and responsive dashboard
